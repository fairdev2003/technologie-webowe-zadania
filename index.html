Prosta Strona HTML

Przykład struktury HTML z różnymi elementami blokowymi, liniowymi, listami, tabelami oraz osadzonymi multimediami.

Przykłady elementów blokowych i liniowych

Element blokowy - <div>

Tekst wewnątrz elementu <div> jest wyświetlany na nowej linii.

Element liniowy - <span>

Element <span> jest elementem liniowym, co oznacza, że nie łamie linii tekstu, w którym się znajduje.
